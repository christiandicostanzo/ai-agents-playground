# AI Agents Playground

Python playground for testing and experimenting with AI agent frameworks.

## Frameworks

- **LangChain**: Framework for building LLM applications
- **Strands**: Multi-agent orchestration framework
- **AutoGen**: Microsoft's multi-agent conversation framework

## Structure

```
├── langchain/     # LangChain experiments
├── strands/       # Strands experiments
├── autogen/       # AutoGen experiments
└── common/        # Shared utilities and configs
```

## Setup

```bash
pip install -r requirements.txt
```
